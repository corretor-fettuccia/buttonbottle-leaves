✨ BtnBottleLeaves

v3.0.0 — Plugin Declarativo

Botões com efeito de vidro líquido e folhas douradas animadas.
Basta adicionar data-bottle-leaves ao seu botão e configurar via atributos HTML.

📦 1. Instalação

Inclua os arquivos da biblioteca no seu projeto:

📁 CSS (no <head>)

<link rel="stylesheet" href="https://corretor-fettuccia.github.io/buttonbottle-leaves/style.css">

📁 JavaScript (antes do </body>)

<script src="https://corretor-fettuccia.github.io/buttonbottle-leaves/js/script.js"></script>

🚀 2. Uso Básico

Adicione o atributo data-bottle-leaves a qualquer botão.

📝 Código HTML

<button data-bottle-leaves style="width: 320px; height: 85px;"></button>

✅ O plugin aplica automaticamente o texto padrão "RESPLANDOR" e 95 folhas.

⚙️ 3. Atributos de Configuração

Personalize cada botão com atributos HTML diretos:
data-bottle-leaves data-texto="Meu Texto" data-folhas="90" data-intensidade="1.2" data-gravidade="0.02" data-herois="0.3"

📝 Exemplos de Configuração

<!-- Botão intenso com muitas folhas -->
<button data-bottle-leaves 
        data-texto="FOGO INTERIOR" 
        data-folhas="120" 
        data-intensidade="1.3">
</button>

<!-- Botão suave e delicado -->
<button data-bottle-leaves 
        data-texto="BRISA" 
        data-folhas="60" 
        data-intensidade="0.8" 
        data-herois="0.2">
</button>

🔄 5. Botões Dinâmicos

Para botões criados após o carregamento da página, use BtnBottleLeaves.scan()

📝 Código para Botões Dinâmicos

// Criar botão dinamicamente
const novoBotao = document.createElement('button');
novoBotao.setAttribute('data-bottle-leaves', '');
novoBotao.setAttribute('data-texto', 'DINÂMICO');
novoBotao.style.width = '320px';
novoBotao.style.height = '85px';
container.appendChild(novoBotao);

// Escanear para ativar o plugin
BtnBottleLeaves.scan();

🎨 6. Estilização Personalizada

Você pode definir width e height diretamente no botão.

📝 Personalizando Tamanho

<button data-bottle-leaves 
        data-texto="MEU BOTÃO" 
        style="width: 450px; height: 100px;">
</button>
