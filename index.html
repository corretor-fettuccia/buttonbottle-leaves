✨ BtnBottleLeaves

v3.2.0 — Plugin Declarativo

Botões com efeito de vidro líquido e folhas douradas animadas.
Basta adicionar data-bottle-leaves ao seu botão e configurar via atributos HTML.
✨ NOVIDADE: Suporte a textos infinitos no hover!

📦 1. Instalação

Inclua os arquivos da biblioteca no seu projeto. São apenas dois arquivos!

🎨 CSS (no <head>)

<link rel="stylesheet" href="https://corretor-fettuccia.github.io/buttonbottle-leaves/style.css">

⚡ JavaScript (antes do </body>)

<script src="https://corretor-fettuccia.github.io/buttonbottle-leaves/js/script1.2.js"></script>

🚀 2. Uso Básico

Adicione o atributo data-bottle-leaves a qualquer botão. O plugin faz o resto!

📝 HTML Mínimo

<button data-bottle-leaves style="width: 320px; height: 85px;"></button>

✅ Resultado: Botão com texto "RESPLANDOR" e 95 folhas douradas animadas.

🔄 3. Múltiplos Textos no Hover INFINITO!

Use data-texto, data-texto1, data-texto2, data-texto3... quantos quiser!
Ao passar o mouse, os textos alternam suavemente a cada 600ms.

📝 Código com 3 textos

<button data-bottle-leaves 
        data-texto="BRISA" 
        data-texto1="LEVE" 
        data-texto2="GOSTOSA">
</button>

📝 Código com 6 textos (infinito!)

<button data-bottle-leaves 
        data-texto="🌅 AMANHECER"
        data-texto1="☀️ MANHÃ"
        data-texto2="🌤️ TARDE"
        data-texto3="🌙 ENTARDECER"
        data-texto4="⭐ NOITE"
        data-texto5="🌌 MADRUGADA">
</button>

💡 Dica: Você pode usar quantos textos quiser! data-texto, data-texto1, data-texto2, data-texto3... até data-texto99

⚙️ 4. Atributos de Configuração

Personalize cada botão com atributos HTML diretos:
data-bottle-leaves data-texto="..." data-texto1="..." data-texto2="..." data-texto3="..." data-folhas="90" data-intensidade="1.2" data-gravidade="0.02" data-herois="0.3"

📝 Exemplos Avançados

<!-- Botão intenso com muitas folhas -->
<button data-bottle-leaves 
        data-texto="FOGO INTERIOR" 
        data-folhas="120" 
        data-intensidade="1.3">
</button>

<!-- Botão suave com texto duplo -->
<button data-bottle-leaves 
        data-texto="BRISA" 
        data-texto1="LEVE"
        data-folhas="60" 
        data-intensidade="0.8" 
        data-herois="0.2">
</button>

📖 5. Referência Completa

Atributo Descrição Padrão Exemplo
data-bottle-leaves Ativa o plugin no botão — obrigatório
data-texto Texto principal (primeiro) "RESPLANDOR" data-texto="OURO"
data-texto1,2,3... Textos alternativos (hover) — data-texto1="BRILHO"
data-folhas Quantidade de folhas (30-200) 95 data-folhas="120"
data-intensidade Força da animação (0.5-2.0) 1.15 data-intensidade="1.5"
data-gravidade Gravidade das folhas (0.01-0.05) 0.02 data-gravidade="0.025"
data-herois Proporção de folhas especiais (0.1-0.6) 0.32 data-herois="0.4"

📐 6. Estilização Personalizada

Você pode definir width e height diretamente no botão via style ou CSS.

📝 Personalizando tamanho

<button data-bottle-leaves 
        data-texto="MEU BOTÃO" 
        data-texto1="TEXTO 2"
        data-texto2="TEXTO 3"
        style="width: 450px; height: 100px;">
</button>

🔄 7. Botões Dinâmicos

Para botões criados após o carregamento da página, use BtnBottleLeaves.scan()

📝 Código para Botões Dinâmicos

// Criar botão dinamicamente
const novoBotao = document.createElement('button');
novoBotao.setAttribute('data-bottle-leaves', '');
novoBotao.setAttribute('data-texto', 'DINÂMICO');
novoBotao.setAttribute('data-texto1', 'CRIADO');
novoBotao.setAttribute('data-texto2', 'AGORA');
novoBotao.style.width = '320px';
novoBotao.style.height = '85px';
container.appendChild(novoBotao);

// Escanear para ativar o plugin
BtnBottleLeaves.scan();

✨ 8. Múltiplos Botões na Mesma Página

Use quantos botões quiser — cada um é independente e tem sua própria animação!
