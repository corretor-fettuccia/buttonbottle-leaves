✨ BtnBottleLeaves

v3.3.0 — Plugin Declarativo

Botões com efeito de vidro líquido e folhas douradas animadas.
Basta adicionar data-bottle-leaves ao seu botão e configurar via atributos HTML.
🎬 NOVIDADE: Controle total sobre timing e estilo das transições de texto!

📦 1. Instalação

Inclua os arquivos da biblioteca no seu projeto. São apenas dois arquivos!

🎨 CSS (no <head>)

<link rel="stylesheet" href="https://corretor-fettuccia.github.io/buttonbottle-leaves/style.css">

⚡ JavaScript (antes do </body>)

<script src="https://corretor-fettuccia.github.io/buttonbottle-leaves/js/script3.5.js"></script>

🚀 2. Uso Básico

Adicione o atributo data-bottle-leaves a qualquer botão. O plugin faz o resto!

📝 HTML Mínimo

<button data-bottle-leaves style="width: 320px; height: 85px;"></button>

✅ Resultado: Botão com texto "RESPLANDOR" e 95 folhas douradas animadas.

🎬 3. Controle de Transição de Texto NOVO!

Agora você controla a velocidade, a suavidade e o estilo da transição entre os textos!
data-transicao-velocidade="ms" data-transicao-suavidade="ms" data-transicao-estilo="fade|slide|scale"

🎥 Exemplo: Transição Cinematográfica (lenta e elegante)

<button data-bottle-leaves 
        data-texto="✨ BRISA ✨" 
        data-texto1="🌊 LEVE" 
        data-texto2="🍃 GOSTOSA"
        data-texto3="🌸 FRESCA"
        data-transicao-velocidade="1200"
        data-transicao-suavidade="500"
        data-transicao-estilo="fade"
        style="width: 380px; height: 95px;">
</button>

💡 Efeito: Transição lenta (1200ms entre textos) com fade suave (500ms) — ideal para mensagens que querem ser contempladas.

✨ 5. Estilos de Transição

Três estilos diferentes para a animação do texto:

📝 Estilos disponíveis

<!-- Fade: desaparece e aparece suavemente -->
data-transicao-estilo="fade"

<!-- Slide: desliza para cima ao trocar -->
data-transicao-estilo="slide"

<!-- Scale: encolhe e expande com fade -->
data-transicao-estilo="scale"

⚙️ 6. Todos os Atributos de Configuração

Personalize cada botão com atributos HTML diretos:
data-bottle-leaves data-texto="..." data-texto1="..." data-texto2="..." data-texto3="..." data-folhas="90" data-intensidade="1.2" data-gravidade="0.02" data-herois="0.3" data-transicao-velocidade="ms" data-transicao-suavidade="ms" data-transicao-estilo="fade|slide|scale"

🔄 10. Botões Dinâmicos

Para botões criados após o carregamento da página, use BtnBottleLeaves.scan()

📝 Código para Botões Dinâmicos

const novoBotao = document.createElement('button');
novoBotao.setAttribute('data-bottle-leaves', '');
novoBotao.setAttribute('data-texto', 'DINÂMICO');
novoBotao.setAttribute('data-texto1', 'CRIADO');
novoBotao.setAttribute('data-texto2', 'AGORA');
novoBotao.setAttribute('data-transicao-velocidade', '800');
novoBotao.setAttribute('data-transicao-suavidade', '300');
novoBotao.setAttribute('data-transicao-estilo', 'slide');
novoBotao.style.width = '320px';
novoBotao.style.height = '85px';
container.appendChild(novoBotao);
BtnBottleLeaves.scan();
